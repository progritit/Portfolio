## 🎨 Project Overview & Design Identity

The goal of this project was to build a fully responsive, professional portfolio website to anchor my CV, LinkedIn, and GitHub profiles. 

Rather than sticking to a generic template, I wanted the design to reflect a personal aesthetic: a **solarpunk-inspired visual identity** mixed with **subtle cosmic and spiritual elements**. The interface balances deep navy and cosmic background tones with vibrant solar gold accents and soft green details. 

Using organic assets created with Gemini—like custom leaf and minimal orbit decorations—the portfolio is designed to feel calm, creative, and purposeful. To truly master front-end fundamentals, the website is intentionally built using **vanilla web technologies without frameworks**.

## 💡 What I Learned

Building a complete project without the safety net of frameworks was an incredible exercise in core engineering. My biggest takeaways include:
*   **Deepening DOM Control:** Managing the state of mobile navigation and interactive elements manually with vanilla JS gave me a fundamental understanding of how the browser works under the hood.
*   **Clean CSS Architecture:** Utilizing CSS variables allowed me to maintain a complex, highly specific color palette (cosmic/solarpunk) without letting the stylesheet turn into spaghetti code.
*   **Production Workflows:** Moving from local Git branches to a live, hosted environment on GitHub Pages forced me to think about asset paths and deployment readiness early on.
